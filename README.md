# ポン出し Pad

スマホで音源を読み込んで、パッドをタップするとすぐ鳴らせるサウンドボードです。

## 使い方

1. スマホで公開URLを開く
2. 「音源追加」から音声ファイルを選ぶ
3. パッドをタップして再生
4. 必要ならパッド右上のボタンから名前、色、音量、再生モードを編集

音源と設定は、開いているスマホのブラウザ内に保存されます。

## Macで動かす

`index.html` を直接開く `file://` 方式だと、ブラウザによって音源保存や一部の機能が制限されます。

Macでは `start.command` をダブルクリックして、開いたブラウザの `http://localhost:4173/` で使ってください。

## GitHub + Netlifyで公開

このプロジェクトはビルド不要の静的サイトです。

Netlifyでは以下の設定で公開できます。

- Build command: 空欄
- Publish directory: `.`

## ファイル構成

- `index.html`: 画面
- `styles.css`: デザイン
- `app.js`: パッド操作と音源保存
- `manifest.json`: ホーム画面追加用の設定
- `sw.js`: オフライン起動用のキャッシュ
- `netlify.toml`: Netlify設定
